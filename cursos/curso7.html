Curso 5: Entorno del Lenguaje de Programación

2.1 Fundamentos conceptuales de programación

Incluyen conceptos como datos, información, lenguajes de alto nivel, tipos de datos, lenguajes de aplicación, compiladores, ensambladores, utilerías de archivos y bibliotecas.

Ejemplo en C
#include <stdio.h> int main() { int numero = 10; // dato printf("El número es: %d", numero); return 0; }
Ejemplo en Python
numero = 10 # dato print("El número es:", numero)

2.2 Estructura de un programa básico

Ejemplo en C
#include <stdio.h> int main() { // Inicio del programa printf("Hola mundo en C\n"); return 0; // Fin del programa }
Ejemplo en Python
# Inicio del programa print("Hola mundo en Python") # Fin del programa

2.3 Declaración de variables

Ejemplo en C
#include <stdio.h> int main() { int edad = 20; float altura = 1.75; printf("Edad: %d, Altura: %.2f", edad, altura); return 0; }
Ejemplo en Python
edad = 20 altura = 1.75 print("Edad:", edad, "Altura:", altura)

2.4 Sentencias de entrada/salida de datos

Ejemplo en C
#include <stdio.h> int main() { int numero; printf("Ingresa un número: "); scanf("%d", &numero); printf("Ingresaste: %d", numero); return 0; }
Ejemplo en Python
numero = input("Ingresa un número: ") print("Ingresaste:", numero)

2.5 Expresiones (Aritméticas, lógicas y relacionales)

Ejemplo en C
#include <stdio.h> int main() { int a = 5, b = 3; printf("Suma: %d\n", a + b); printf("Comparación (a > b): %d\n", a > b); printf("AND lógico: %d\n", (a > 0 && b > 0)); return 0; }
Ejemplo en Python
a = 5 b = 3 print("Suma:", a + b) print("Comparación (a > b):", a > b) print("AND lógico:", (a > 0 and b > 0))

2.6 Depuración de un programa

La depuración consiste en identificar y corregir errores. Se utilizan mensajes de impresión o depuradores.

Ejemplo en C
#include <stdio.h> int main() { int x = 10; printf("Valor de x: %d\n", x); // depuración con printf return 0; }
Ejemplo en Python
x = 10 print("Valor de x:", x) # depuración con print

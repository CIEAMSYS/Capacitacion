Programación en Python y C

Guía sobre los fundamentos de programación, utilizando los lenguajes de Python y C, con ejemplos prácticos y elementos interactivos.

2.1. Fundamentos Conceptuales de Programación

Datos vs. Información

Dato: Valor crudo y sin procesar (ej: 25, 'A', True).

Información: Resultado de procesar y contextualizar datos para darles significado (ej: "La temperatura es de 25°C").

Lenguajes de Programación

Python: Lenguaje de alto nivel con sintaxis clara y legible. Prioriza la productividad del desarrollador.

C: Lenguaje de alto nivel de sistemas. Ofrece estructuras de alto nivel pero permite control de bajo nivel sobre memoria y hardware.

Tipos de Datos

Tipo Python C
Entero int (tipado dinámico) int, short, long (tipado estático)
Flotante float float, double
Carácter str (de 1 carácter) char
Cadena str char[] (array de caracteres)
Booleano bool bool (C99) o int

Compiladores vs. Intérpretes

C: Lenguaje compilado. Se traduce completamente a código máquina generando un ejecutable.

Python: Lenguaje interpretado. El código se ejecuta línea por línea por un intérprete.

Bibliotecas

C: Colecciones de funciones precompiladas. Se incluyen con #include.

Python: Módulos o paquetes. Se incluyen con import.

2.2. Estructura de un Programa Básico

Estructura en C

Estructura básica de un programa en C 
#include <stdio.h>

int main() {
    // Cuerpo del programa
    printf("Hola, Mundo!\n");
    
    return 0;
}

Estructura en Python

Estructura básica de un programa en Python 
# Importación de módulos (opcional)
import math

# Función principal (convención, no obligatoria)
def main():
    # Cuerpo del programa
    print("Hola, Mundo!")

# Punto de entrada del script
if __name__ == "__main__":
    main()

2.3. Declaración de Variables

Declaración en C (Estática y Explícita)

Declaración de variables en C 
int edad = 25;          // Entero
float precio = 19.99;   // Flotante
char letra = 'A';       // Carácter
char nombre[] = "Ana";  // Cadena de caracteres

Declaración en Python (Dinámica e Implícita)

Declaración de variables en Python 
edad = 25          # Se infiere que es un int
precio = 19.99     # Se infiere que es un float
letra = 'A'        # Se infiere que es un str (de un solo carácter)
nombre = "Ana"     # Se infiere que es un str

2.4. Sentencias de Entrada/Salida de Datos

Entrada/Salida en C

Entrada y salida en C 
#include <stdio.h>

int main() {
    int edad;
    float altura;
    char nombre[50];
    
    // Entrada
    printf("Introduce tu nombre: ");
    scanf("%s", nombre);
    
    printf("Introduce tu edad: ");
    scanf("%d", &edad);
    
    printf("Introduce tu altura: ");
    scanf("%f", &altura);
    
    // Salida
    printf("Hola %s, tienes %d años y mides %.2f metros.\n", nombre, edad, altura);
    
    return 0;
}

Entrada/Salida en Python

Entrada y salida en Python 
# Entrada
nombre = input("Introduce tu nombre: ")
edad = int(input("Introduce tu edad: "))
altura = float(input("Introduce tu altura: "))

# Salida
print(f"Hola {nombre}, tienes {edad} años y mides {altura:.2f} metros.")

2.5. Expresiones (Aritméticas, Lógicas y Relacionales)

Operadores en C y Python

Categoría Operadores (C y Python) Ejemplo en C Ejemplo en Python
Aritméticos +, -, *, /, % int res = 5 + 3 * 2; res = 5 + 3 * 2
Relacionales ==, !=, >, <, >=, <= if (edad >= 18) { ... } if edad >= 18:
Lógicos && (and), || (or), ! (not) if (x > 0 && x < 10) if x > 0 and x < 10:

Ejemplo de Expresiones en C

Expresiones en C 
#include <stdio.h>

int main() {
    int a = 10, b = 5;
    int suma = a + b;
    int resta = a - b;
    int multiplicacion = a * b;
    float division = a / (float)b; // Conversión a float
    
    // Expresiones relacionales
    int mayor = a > b; // 1 (true)
    int igual = a == b; // 0 (false)
    
    // Expresiones lógicas
    int condicion = (a > 0) && (b < 10); // 1 (true)
    
    printf("Suma: %d\n", suma);
    printf("División: %.2f\n", division);
    printf("¿A > B? %d\n", mayor);
    printf("¿A > 0 y B < 10? %d\n", condicion);
    
    return 0;
}

2.6. Depuración de un Programa

Errores Comunes

En C:

  • Errores de sintaxis (olvidar ;, {}, ())
  • Errores de tiempo de ejecución (acceso a memoria incorrecta)
  • Errores lógicos (lógica incorrecta en bucles o condiciones)

En Python:

  • Errores de sintaxis (indentación, olvidar :)
  • Errores de tiempo de ejecución (TypeError, NameError, IndexError)
  • Errores lógicos (similares a C)

Técnicas de Depuración

En C: Compiler warnings, GDB (GNU Debugger), prints de depuración.

En Python: pdb (Python Debugger), IDEs con depuradores visuales, prints de depuración.

Ejemplo de Depuración en C

Depuración en C con printf 
#include <stdio.h>

int main() {
    int valores[5] = {10, 20, 30, 40, 50};
    int suma = 0;
    
    printf("Depuración: Iniciando bucle...\n");
    
    for (int i = 0; i < 5; i++) {
        printf("Depuración: i = %d, valores[%d] = %d\n", i, i, valores[i]);
        suma += valores[i];
        printf("Depuración: suma parcial = %d\n", suma);
    }
    
    printf("Suma total: %d\n", suma);
    
    return 0;
}

Ejemplo de Depuración en Python

Depuración en Python con print 
valores = [10, 20, 30, 40, 50]
suma = 0

print("Depuración: Iniciando bucle...")

for i, valor in enumerate(valores):
    print(f"Depuración: i = {i}, valor = {valor}")
    suma += valor
    print(f"Depuración: suma parcial = {suma}")

print(f"Suma total: {suma}")

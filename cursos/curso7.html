Entorno del Lenguaje de Programación

2.1 Fundamentos conceptuales de programación

La programación consiste en diseñar y construir programas que resuelvan problemas. Conceptos básicos:

  • Datos: valores que manipula un programa.
  • Información: datos procesados con un significado.
  • Lenguajes de alto nivel: como C o Python, fáciles de entender por humanos.
  • Tipos de datos: enteros, flotantes, caracteres, cadenas, booleanos.
  • Compiladores y ensambladores: traducen el código a lenguaje máquina.
  • Bibliotecas: colecciones de funciones listas para usar.

Ejemplo en C

#include <stdio.h>
int main() {
    int numero = 10;   // dato tipo entero
    printf("El número es: %d\n", numero);
    return 0;
}

Ejemplo en Python

numero = 10   # dato tipo entero
print("El número es:", numero)

2.2 Estructura de un programa

Un programa tiene una estructura básica: declaraciones, instrucciones y salida de resultados.

Ejemplo en C

#include <stdio.h>
int main() {
    printf("Hola, mundo!\n");
    return 0;
}

Ejemplo en Python

print("Hola, mundo!")

2.3 Declaración de variables

Las variables almacenan datos que se usan en un programa.

Ejemplo en C

#include <stdio.h>
int main() {
    int edad = 20;
    float altura = 1.75;
    char inicial = 'A';
    printf("Edad: %d, Altura: %.2f, Inicial: %c\n", edad, altura, inicial);
    return 0;
}

Ejemplo en Python

edad = 20
altura = 1.75
inicial = "A"
print("Edad:", edad, "Altura:", altura, "Inicial:", inicial)

2.4 Sentencias de entrada/salida de datos

Permiten interactuar con el usuario.

Ejemplo en C

#include <stdio.h>
int main() {
    int numero;
    printf("Ingrese un número: ");
    scanf("%d", &numero);
    printf("El número ingresado fue: %d\n", numero);
    return 0;
}

Ejemplo en Python

numero = int(input("Ingrese un número: "))
print("El número ingresado fue:", numero)

2.5 Expresiones (Aritméticas, lógicas y relacionales)

Las expresiones combinan valores y operadores para producir un resultado.

Ejemplo en C

#include <stdio.h>
int main() {
    int a = 5, b = 3;
    printf("Suma: %d\n", a + b);        // aritmética
    printf("Mayor: %d\n", a > b);       // relacional
    printf("Condición: %d\n", (a > 0 && b > 0)); // lógica
    return 0;
}

Ejemplo en Python

a, b = 5, 3
print("Suma:", a + b)          # aritmética
print("Mayor:", a > b)         # relacional
print("Condición:", a > 0 and b > 0)  # lógica

2.6 Depuración de un programa

La depuración es el proceso de encontrar y corregir errores en el código.

Ejemplo en C

#include <stdio.h>
int main() {
    int x = 10;
    int y = 0;
    // printf("%d", x / y); // Error: división por cero
    printf("Depuración: evitar errores como división entre cero\n");
    return 0;
}

Ejemplo en Python

x = 10
y = 0
# print(x / y)  # Error: división por cero
print("Depuración: evitar errores como división entre cero")

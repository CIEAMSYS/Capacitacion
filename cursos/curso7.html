Capítulo 1: Introducción a C

      
El lenguaje C es un lenguaje de programación de propósito general desarrollado en 1972 por Dennis Ritchie en los Laboratorios Bell. Es uno de los lenguajes más utilizados y ha influido en muchos otros lenguajes como C++, Java y C#.

      
Características principales:

      
    
        
  • Lenguaje estructurado
    • 
        
  • De bajo nivel (acceso directo a memoria)
    • 
        
  • Portable (código puede ejecutarse en diferentes plataformas)
    • 
        
  • Eficiente y rápido
    • 
      

      

        Ejemplo: Hola Mundo

        #include <stdio.h>

        int main() {

          printf("¡Hola, mundo!\n");

          return 0;

        }
      

    

    
    
    

      
Capítulo 2: Variables y Tipos de Datos

      
En C, las variables son contenedores para almacenar datos. Cada variable tiene un tipo específico que determina el tamaño y formato de los datos que puede contener.

      
Tipos de datos básicos en C:

      
    
        
  • int: Números enteros
    • 
        
  • float: Números decimales de precisión simple
    • 
        
  • double: Números decimales de precisión doble
    • 
        
  • char: Caracteres individuales
    • 
      

      

        Ejemplo: Declaración de variables

        #include <stdio.h>

        int main() {

          int edad = 25;

          float altura = 1.75;

          char inicial = 'J';

          printf("Edad: %d, Altura: %.2f, Inicial: %c\n", edad, altura, inicial);

          return 0;

        }
      

    

    
    
    

      
Capítulo 3: Estructuras de Control

      
Las estructuras de control permiten dirigir el flujo de ejecución de un programa. En C tenemos:

      
    
        
  • if/else: Para tomar decisiones
    • 
        
  • switch: Para selección múltiple
    • 
        
  • for: Para bucles con contador
    • 
        
  • while: Para bucles con condición al inicio
    • 
        
  • do-while: Para bucles con condición al final
    • 
      

      

        Ejemplo: Estructuras if y for

        #include <stdio.h>

        int main() {

          int i;

          for(i = 1; i <= 5; i++) {

            if(i % 2 == 0) {

              printf("%d es par\n", i);

            } else {

              printf("%d es impar\n", i);

            }

          }

          return 0;

        }
      

    

    
    
    

      
Capítulo 4: Funciones

      
Las funciones permiten dividir un programa en partes más pequeñas y manejables. Cada función realiza una tarea específica y puede ser reutilizada.

      
Partes de una función:

      
    
        
  • Declaración: Especifica el nombre, tipo de retorno y parámetros
    • 
        
  • Definición: Implementa lo que hace la función
    • 
        
  • Llamada: Ejecuta la función
    • 
      

      

        Ejemplo: Función para calcular factorial

        #include <stdio.h>

        // Declaración de la función

        int factorial(int n);

        

        int main() {

          int num = 5;

          printf("El factorial de %d es %d\n", num, factorial(num));

          return 0;

        }

        

        // Definición de la función

        int factorial(int n) {

          if(n == 0) return 1;

          return n * factorial(n-1);

        }
      

    

    
    
    

      
Capítulo 5: Arreglos y Cadenas

      
Los arreglos son estructuras de datos que almacenan múltiples valores del mismo tipo. Las cadenas en C son arreglos de caracteres terminados con el carácter nulo '\0'.

      
Características de los arreglos:

      
    
        
  • Elementos del mismo tipo
    • 
        
  • Acceso mediante índices (comenzando en 0)
    • 
        
  • Tamaño fijo (en C estándar)
    • 
      

      

        Ejemplo: Trabajando con arreglos y cadenas

        #include <stdio.h>

        #include <string.h>

        

        int main() {

          int numeros[5] = {10, 20, 30, 40, 50};

          char nombre[20] = "Juan";

          

          printf("Primer número: %d\n", numeros[0]);

          printf("Nombre: %s\n", nombre);

          printf("Longitud del nombre: %zu\n", strlen(nombre));

          

          return 0;

        }
      

    

    
    
    

      
Capítulo 6: Punteros

      
Los punteros son variables que almacenan direcciones de memoria. Son una de las características más poderosas (y complejas) de C.

      
Conceptos clave:

      
    
        
  • &: Operador de dirección (devuelve la dirección de una variable)
    • 
        
  • *: Operador de indirección (accede al valor en una dirección)
    • 
        
  • NULL: Valor especial que indica que un puntero no apunta a nada
    • 
      

      

        Ejemplo: Uso básico de punteros

        #include <stdio.h>

        

        int main() {

          int x = 10;

          int *ptr = &x;  // ptr apunta a x

          

          printf("Valor de x: %d\n", x);

          printf("Dirección de x: %p\n", &x);

          printf("Valor de ptr: %p\n", ptr);

          printf("Valor al que apunta ptr: %d\n", *ptr);

          

          *ptr = 20;  // Cambiamos el valor de x a través del puntero

          printf("Nuevo valor de x: %d\n", x);

          

          return 0;

        }
      

    

    
    
    

      
Capítulo 7: Estructuras y Uniones

      
Las estructuras (struct) permiten agrupar variables de diferentes tipos bajo un mismo nombre. Las uniones (union) son similares pero comparten la misma ubicación de memoria.

      
Diferencias entre struct y union:

      
    
        
  • Struct: Cada miembro tiene su propia ubicación de memoria
    • 
        
  • Union: Todos los miembros comparten la misma ubicación de memoria
    • 
      

      

        Ejemplo: Uso de estructuras

        #include <stdio.h>

        #include <string.h>

        

        // Definición de una estructura

        struct Persona {

          char nombre[50];

          int edad;

          float altura;

        };

        

        int main() {

          // Declaración e inicialización de una estructura

          struct Persona p1;

          strcpy(p1.nombre, "María");

          p1.edad = 30;

          p1.altura = 1.65;

          

          printf("Nombre: %s\n", p1.nombre);

          printf("Edad: %d\n", p1.edad);

          printf("Altura: %.2f\n", p1.altura);

          

          return 0;

        }
